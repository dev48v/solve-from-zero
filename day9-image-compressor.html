🔨 Build it yourself

    
FileReader → Image → Canvas → toBlob. ~30 lines, no libraries.

    
    
      
  1. 1
    Catch the dropped file
    drop.addEventListener("drop", e => {
  e.preventDefault();
  const file = e.dataTransfer.files[0];   // never uploaded — stays local
  loadImage(file);
});
    2. 
      
  2. 2
    Read it into an <img>
    const img = new Image();
img.src = URL.createObjectURL(file);   // a local blob: URL
await img.decode();
    3. 
      
  3. 3
    Draw it onto a canvas
    canvas.width = img.width; canvas.height = img.height;
canvas.getContext("2d").drawImage(img, 0, 0);
    4. 
      
  4. 4
    Re-encode at a quality (the compression)
    canvas.toBlob(blob => {
  showSize(blob.size);
  preview.src = URL.createObjectURL(blob);
}, "image/jpeg", 0.7);   // 0.7 = 70% quality
    5. 
      
  5. 5
    Offer the blob as a download
    link.href = URL.createObjectURL(blob);
link.download = "compressed.jpg";   // browser saves it, no server
