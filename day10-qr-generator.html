🔨 Build it yourself

    
QR encoding (Reed-Solomon, masking) is genuinely hard — so use a tiny battle-tested library and focus on the UX. ~20 lines.

    
    
      
  1. 1
    Pull in a QR library
    <script src="https://cdn.jsdelivr.net/npm/qrcode@1.5.3/build/qrcode.min.js"></script>
// implementing Reed-Solomon error correction yourself is a rabbit hole —
// this is the right place to stand on a library's shoulders.
    2. 
      
  2. 2
    Render to a canvas
    QRCode.toCanvas(canvas, text, {
  width: 240,
  errorCorrectionLevel: "M",
  color: { dark: "#0f172a", light: "#ffffff" },
});
    3. 
      
  3. 3
    Regenerate live on every change
    [input, eccSelect, colorPicker].forEach(el =>
  el.addEventListener("input", generate));
    4. 
      
  4. 4
    Download the canvas as PNG
    link.href = canvas.toDataURL("image/png");
link.download = "qr.png";   // pure client-side, no server
    5. 
      
  5. 5
    Special payloads = standard strings
    // a wifi QR is just a formatted string the camera knows:
`WIFI:S:${ssid};T:WPA;P:${password};;`
