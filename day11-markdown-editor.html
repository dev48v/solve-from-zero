🔨 Build it yourself

    
Escape HTML, then transform line-by-line + inline regex. ~40 lines for a usable subset.

    
    
      
  1. 1
    Escape HTML FIRST (security)
    md = md.replace(/&/g,"&").replace(//g,">");
// so user input can't inject <script> — the #1 rule of any renderer
    2. 
      
  2. 2
    Pull out fenced code blocks first
    md = md.replace(/```([\s\S]*?)```/g, (_, c) => `<pre><code>${c}</code></pre>`);
// do these before other rules so their contents aren't re-parsed
    3. 
      
  3. 3
    Block elements line-by-line
    line.replace(/^### (.*)/, "<h3>$1</h3>")
    .replace(/^## (.*)/,  "<h2>$1</h2>")
    .replace(/^# (.*)/,   "<h1>$1</h1>")
    .replace(/^> (.*)/,   "<blockquote>$1</blockquote>");
    4. 
      
  4. 4
    Inline styles with regex
    .replace(/\*\*(.+?)\*\*/g, "<strong>$1</strong>")
.replace(/\*(.+?)\*/g,     "<em>$1</em>")
.replace(/`(.+?)`/g,       "<code>$1</code>")
.replace(/\[(.+?)\]\((.+?)\)/g, '<a href="$2">$1</a>');
