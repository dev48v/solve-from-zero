🛣️ Pothole Detector from Dashcam

📋 The Problem (Smart India Hackathon brief)

Indian roads have ~9 million potholes annually. Municipal complaints rely on citizens manually photographing each one. Build a system that takes dashcam video, detects potholes per frame, GPS-tags them, and posts to the civic complaint API automatically.

  2. 
        
    2
    Detect in a video file
    
        
    

    from ultralytics import YOLO
import cv2

model = YOLO("pothole-yolov8n.pt")

cap = cv2.VideoCapture("dashcam.mp4")
while cap.isOpened():
    ok, frame = cap.read()
    if not ok: break

    # YOLO runs ~30 FPS on a CPU laptop
    results = model(frame, conf=0.6)
    for box in results[0].boxes:
        x1, y1, x2, y2 = box.xyxy[0]
        conf = float(box.conf)
        cv2.rectangle(frame, (int(x1), int(y1)), (int(x2), int(y2)), (0,0,255), 2)
        cv2.putText(frame, f"pothole {conf:.2f}", (int(x1), int(y1)-5),
                    cv2.FONT_HERSHEY_SIMPLEX, 0.6, (0,0,255), 2)
    cv2.imshow("Detect", frame)
    if cv2.waitKey(1) & 0xFF == 27: break

    Pair frames with GPS
    
        
    Dashcam video usually has a parallel GPX file or embedded GPS metadata. Map each video frame's timestamp to lat/lng. gpxpy for GPX files, exiftool for embedded GoPro telemetry.

    4
    Cluster duplicate detections
    
        
    A single pothole shows in 20-50 consecutive frames as you drive over it. Group detections within a 10-meter radius into ONE complaint. Use a simple `groupby(round(lat, 4), round(lng, 4))`.
