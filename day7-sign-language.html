🤟 Sign Language → Text, From Your Webcam

Show a sign — it reads your hand

No model upload, no cloud. Hand tracking runs in your browser. Hold a sign steady for a moment to "type" it. Recognises counts 0–5 plus 👍 and 🖐 hello.

🤟
Loads a ~7 MB hand-tracking model the first time. Needs camera permission. Chrome/Edge recommended.

🔨 Build it yourself

    
MediaPipe gives you 21 hand points; you turn them into signs with simple geometry. ~60 lines.

    
    
      
  1. 1
    Load the hand landmarker
    import { HandLandmarker, FilesetResolver } from
  "https://cdn.jsdelivr.net/npm/@mediapipe/tasks-vision/vision_bundle.mjs";
const files = await FilesetResolver.forVisionTasks(".../wasm");
const hand = await HandLandmarker.createFromOptions(files, { runningMode:"VIDEO" });

    Detect 21 landmarks each frame
    const res = hand.detectForVideo(video, performance.now());
const lm = res.landmarks[0];   // [{x,y,z}, ...21 points]

    // for index/middle/ring/pinky: tip.y < pip.y  → extended
const up = [8,12,16,20].map((tip,i) => lm[tip].y < lm[[6,10,14,18][i]].y);
