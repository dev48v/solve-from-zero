🔨 Build it yourself

    
The whole thing is the browser's SpeechRecognition API and one event handler. ~30 lines.

    
    
      
  1. 1
    Grab the built-in engine
    const SR = window.SpeechRecognition || window.webkitSpeechRecognition;
const rec = new SR();
rec.lang = "en-US";
    2. 
      
  2. 2
    Turn on continuous + interim results
    rec.continuous = true;       // don't stop after one sentence
rec.interimResults = true;   // emit live guesses, not just finals
    3. 
      
  3. 3
    Split interim (grey) from final (locked)
    rec.onresult = e => {
  let interim = "";
  for (let i = e.resultIndex; i < e.results.length; i++) {
    const r = e.results[i];
    if (r.isFinal) finalText += r[0].transcript;
    else interim += r[0].transcript;
  }
  render(finalText, interim);
};
    4. 
      
  4. 4
    Auto-restart when it stops
    rec.onend = () => { if (listening) rec.start(); };
// the engine times out on silence — restart to keep going
    5. 
      
  5. 5
    Caption any tab's audio (the real trick)
    // share a tab WITH audio, then feed that stream's audio out
const stream = await navigator.mediaDevices.getDisplayMedia({ audio:true });
